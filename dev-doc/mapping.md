Как дожно вести себя меню и кнопки:

---

После /start пользователь видит:
Reply кнопки по 1 шт в ряд (Добавить источник, Мои источники, Настройки)

Добавить источник - добавляем источник данных
Мои источники - выводит список источников. При нажатии на источник показывает его виджеты с состоянием (inline кнопки вкл/выкл), Снизу кноки "удалить" и "назад" (к источникам) пример:
НТЭЦ
Состояние источника (ВКЛ / ВЫКЛ)
Виджет1 - вкл
Виджет2 - выкл
Удалить (для удаления требуется подтверждение)
Назад

---

При нажатии на "Добавить источник" бот отправляет в чат сообщения: 
1. Введите название источника: (Пользователь задает имя источнику отправляя сообщение в чат)
2. Введите ID компании: (Пользователь задает ID компании отправляя сообщение в чат)
3. Введите API Base URL:
(например: https://helmetsapi.diwo.tech): (Пользователь задает API Base URL компании отправляя сообщение в чат)
4. Введите Auth Base URL:
(например: https://helmetsauth.diwo.tech): (Пользователь задает Auth Base URL компании отправляя сообщение в чат)
5. Введите логин: (Пользователь задает логин учетной записи отправляя сообщение в чат)
6. Введите пароль: (Пользователь задает пароль учетной записи отправляя сообщение в чат)
7. Введите частоту опроса (от 60 до 86400 секунд): (Пользователь задает частоту опроса источника API отправляя сообщение в чат)

В случае ошибки выводит сообщение ошибки:

⚠️ Проверка не удалась: Client error '422 Unprocessable Entity' for url 'https://helmetsauth.diwo.tech/1.0/Auth/SignInByEmail' 

БЕЗ: For more information check: https://developer.mozilla.org/en-US/docs/Web/HTTP/Status/422

Не добавляет источник и не предлагает настройки.

В случае успеха выводит тестовую краткую сводку:

и предлагает настройки виджетов в inline кнопках:
1. Резкое падение
2. Сводка - убрать! Сводка глобальный виджет для всех источников, настраивается через "настройки", по умолчанию выключена. При нажатии на "Сводка" в настройка в чат отправляется сообщение: Введите время сводки (ЧЧ:ММ) и inline кнопка "ВКЛ / ВЫКЛ"
Сейчас: 08:00 и пользователь указывает нужное время отправкой в чат.
3. Готово (завершает настройку источника)
